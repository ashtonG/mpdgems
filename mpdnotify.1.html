NAME

mpdnotify - stupid desktop notifier for MPD

SYNOPSIS

mpdnotify [OPTIONS]

DESCRIPTION

mpdnotify is a simple daemon that shows a desktop notification when the song that MPD is playing changes. It can also optionally show the album cover art if the music database is located on the local system.

OPTIONS

-d, --dir

       The "root" music directory. This is used to find album cover files.

-c, --cover

       The cover file name (default cover.jpg).

-C, --nocover

       Do not insert the cover images in the notifications.

-a, --addr

       The MPD server address.

-p, --port

       The MPD server port.

-s, --secret

       The MPD password.

-h, --help

       Show the help message.

ENVIRONMENT

MPD_HOST

       The MPD server address (overridden by --addr).

MPD_PORT

       The MPD server port (overridden by --port).
